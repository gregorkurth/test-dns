# Requirement: RDTS-712

> **Typ:** [ARCH] Architektur-Anforderung
> **Priorität:** 🟥 MUSS
> **Service Function:** SFN-SCS-004 – Release Artifact Audit
> **Capability:** CAP-007 Supply Chain Security

## Anforderungstext

Die Freigabe des Artefaktinhalts erfolgt über eine versionierte Allowlist-Richtlinie mit explizit freigegebenen Dateien, Pfaden und Ausnahmen; eine reine Ignore- oder Blocklist-Strategie genügt nicht.
