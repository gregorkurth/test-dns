# Service Function: Release Artifact Audit (SFN-SCS-004)

> **Service:** SVC-SCS-SCAN – Security Scanning
> **Capability:** CAP-007 Supply Chain Security
> **Quelle:** [ARCH] Architektur-Anforderung

## Beschreibung

Prüft den final erzeugten Veröffentlichungsinhalt vor Publish, Release oder Offline-Export gegen eine versionierte Freigaberichtlinie. Verhindert, dass Secrets, interne Entwicklungsartefakte, unnötige Source-Dateien oder unerlaubte Sourcemaps in freigegebene Artefakte gelangen.
