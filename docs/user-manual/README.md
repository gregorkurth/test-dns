# Benutzerhandbuch

Dieses Handbuch ist fuer Leser gedacht, die einen klaren Einstieg ohne Entwicklerwissen brauchen.

Es ist in drei Themenbereiche gegliedert:

- [Management](management.md)
- [Betrieb](operations.md)
- [Umsetzung](implementation.md)

## Wie lesen?

- Management: wenn du den Nutzen, die Version und den Status verstehen willst
- Betrieb: wenn du die App starten, pruefen oder im Alltag betreiben willst
- Umsetzung: wenn du die technische Struktur und die wichtigsten Abhaengigkeiten sehen willst

## Kurzer Hinweis

Das Benutzerhandbuch ersetzt nicht die technische Doku.
Es fuehrt nur schneller zu den Antworten, die fuer die jeweilige Rolle wichtig sind.
